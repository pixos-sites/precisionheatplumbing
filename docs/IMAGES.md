## Adding New Images

1. Add files to the correct trade folder under `public/images/<trade>/`
2. Update the table above with the new path
3. Preferred format: JPG or WebP
4. Recommended sizes: Hero 1920×1080 · Gallery 1080×1080

## Logo Files

Client logos go in their trade folder or a new `public/images/<clientid>/` folder.
Reference as: `/images/<clientid>/logo.png`
